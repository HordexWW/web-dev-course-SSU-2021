Бинарное слово

Данная функция принимает на вход строку ASCII символов и возвращает массив строк, каждый элемент которого является ASCII-кодом соответствующего символа в двоичном формате.

Игра на выбывание

Данная функция принимает два параметра – количество участников в круге n и шаг k, на котором участник выбывает из круга в ходе одного раунда. Каждый раунд функция исключает одного участника и продолжает определять выбывших по кругу до тех пор, пока круг не сократится до одного победителя.
